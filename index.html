Pizza Parlor

This is a Pizza Parlor Order Page!
This application was created as an independent project at Epicodus to showcase proper use of JavaScript Objects, Constructors, and Prototypes. This project was created using Test Driven Development. Plain text tests can be found in the test section of the README.md. This application allows the user to make multiple selections using radio buttons to create their own pizza. When the user clicks the Order Now button, the total cost of the custom pizza will display.

Mr. Roboger's Neighborhood

This is Mr. Roboger's Neighborhood
This Epicodus independant project was created using Test Driven Development. This application will output a list of whole numbers from 0 to the user input number. It replaces some numbers in the list with strings, following these rules, in order of importance: Rule 1: For numbers that contain a 3, the entire number is replaced with "Won't you be my neighbor?" Rule 2: For numbers that contain a 2, the entire number is replaced with "Boop!" Rule 3: For numbers that contain a 3, the entire number is replaced with "Won't you be my neighbor?"
